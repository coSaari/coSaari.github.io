Reveal.js

Feature Description

Cool Layout Features

Stacking

This feature allows you to center and place elements on top of each other for an interesting display animation.

Ladies and Gentlemen: Jim Carrey!

How to Stack

Use class "r-stack" in conjunction with the "fragment" class to stack elements like images.

Framing

Framing is an easy layout feature to make images or logos pop on the page, or link to other sites.

How to Frame

By using the class "r-frame" in any element, within an anchor, you will be able to utilize this cool feature.
