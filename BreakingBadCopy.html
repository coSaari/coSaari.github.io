Breaking Bad Analysis

Breaking Bad Viewing Analysis

Version 1

  For this first version I created a simple plot showing the comparison between the number of total US viewers per episode of Breaking Bad. This initial visualization uses a simple color scheme to represent the changing seasons of the show and the general trend of linearly increasing ratings.


Version 2

  The second version of this visualization builds off of the original bar chart showing viewers vs. episode of Breaking Bad. The addition to this chart is the overlapping regression lines. These color coded regression lines help the user to see the general trend of viewers per episode, this also is helped with the utilization of a legend to show each regression degree.


Version 3

  The third version of this visualization adds the interactive tool tip element. This tool tip allows the user to hover of the individual episodes and see its corresponding season, episode number, and year it was produced. Each will allow the user to more accurately look into the information behind the general trend of the overall plot.
